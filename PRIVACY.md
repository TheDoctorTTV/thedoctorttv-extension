# Privacy Policy

TheDoctorTTV Extension does not collect, sell, share, or transmit personal data.

The extension stores user settings with Chrome extension storage, including feature toggle states and the configured new tab destination. If Chrome Sync is enabled in the browser, Chrome may sync those settings through the signed-in Google account according to Chrome's own sync settings.

The extension reads page content only on the supported sites declared in `manifest.json` so it can perform local browser features:

- Ko-fi theme sync
- Twitch chat copy button
- X/Twitter share URL rewriting
- X/Twitter promoted post hiding
- YouTube Shorts cleanup

Clipboard access is used only for user-initiated copy actions and local share URL rewriting. Clipboard contents are not sent to any server.
